# University Welcome App

A desktop application built with **Avalonia UI** (.NET 8) for student registration and welcome message generation.

## Features

- Student registration form (name, department, semester)
- Personalized welcome message generation
- Clean, modern UI with professional styling

## Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

## Getting Started

```bash
# Restore dependencies
dotnet restore

# Run the application
dotnet run
```

## Build

```bash
dotnet build -c Release
```

## Tech Stack

- **Framework:** .NET 8
- **UI:** Avalonia UI (cross-platform desktop)
- **Themes:** Fluent
